## Coding Technique Exploration

A p5.js Perlin noise flow field could help create the fluid particle movement. This technique uses noise values to generate invisible directional forces across the canvas, so particles move in smooth, organic paths instead of random, chaotic lines. It can make particles look like flowing data, wave-like colour transitions, and soft digital motion.
